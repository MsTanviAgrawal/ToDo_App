
# ToDo App

This project is a simple and clean ToDo application built to help users keep track of their tasks in an organized way. The goal of this app is to provide an easy and intuitive experience where anyone can add tasks, update them, mark them as completed, and remove them effortlessly.

---

## Overview

The ToDo App allows users to:

* Add new tasks
* Mark tasks as completed
* Edit existing tasks
* Delete tasks
* Maintain a clear and organized task list

It is designed with simplicity in mind, making it perfect for daily use.

---

## Features

* Add, edit, and delete tasks
* Mark tasks as done
* Minimal and user-friendly UI
* Fully responsive design
* Smooth and fast interactions

---

## Tech Stack

* **React JS**
* **JavaScript**
* **CSS / Tailwind / Custom Styling**

## Running the Project Locally

1. Clone the repository:

   ```bash
   git clone https://github.com/MsTanviAgrawal/ToDo_App.git
   ```
2. Navigate into the project folder
3. Install dependencies:

   ```bash
   npm install
   ```
4. Start the server:

   ```bash
   npm start
   ```
5. Open your browser at **[http://localhost:3000/](http://localhost:3000/)**
